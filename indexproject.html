Project By ME

Project ที่นำมานี้เป็น Project เพื่อแนะนำความสามารถในการให้คำปรึกษาเกี่ยว กับภาษา HTML CSS JS และ TS

เพื่อช่วยเหลือการศึกษาผู้ที่ต้องการศึกษาในเรื่องของภาษา IT

ADDON-FATE-Sword
FATE-Sword เป็นระบบเสริมที่เพิ่มทั้งประสิทธิภาพและความสนุกให้กับผู้เล่นใน Minecraft Bedrock Edition โดยนำ modal FATE-Sword มาดัดแปลงให้กลายเป็นอาวุธ พร้อมเพิ่มลูกเล่นต่าง ๆ เช่น ท่าอนิเมชันพิเศษของ FATE-Sword เพื่อให้การเล่นมีความน่าสนใจและตื่นเต้นมากยิ่งขึ้น

  • ระบบส่วนเสริม Minecraft Bedrock Edition
  • การใช้ JS,TS เพื่อช่วยเติมโค้ดและทำอนิเมชั่น
  • การดีบักโค้ด (Debugging) โดยการตั้งเบรกพอยต์และตรวจสอบค่าตัวแปร
  • การใช้งานระบบควบคู่กับระบบ หลายผู้เล่น

ADDON-THIE
addon-Thie เป็นส่วนเสริม (Add-on) สำหรับ Minecraft Bedrock Edition ที่ออกแบบมาเพื่อให้ผู้เล่นสามารถขโมยของจากตัวผู้เล่นคนอื่นได้อย่างสมจริง พร้อมระบบมินิเกมหลากหลายที่ใช้ทักษะและความเร็วในการขโมย รวมถึงอุปสรรคต่าง ๆ ที่ทำให้การขโมยของมีความท้าทายมากขึ้น

  • ระบบขโมยของจากผู้เล่นคนอื่นแบบสมจริง
  • มินิเกมทดสอบทักษะ เช่น งัดกระเป๋า, ปลดล็อกกุญแจ, ลอบเร้น
  • มีอุปสรรคและความท้าทายที่แตกต่างกันในแต่ละครั้ง
  • รองรับการเล่นในโหมด RPG, Survival Multiplayer และ Roleplay
  • ระบบแจ้งเตือนและลงโทษเมื่อถูกจับได้

ADDON-MINE-TAGS-BEV
ADDON-MINE-TAGS-BEV เป็นระบบอาชีพสำหรับ Minecraft Bedrock Edition ที่ใช้ tag ประจำอาชีพเพื่อกำหนดความสามารถของผู้เล่น โดยมีคุณสมบัติดังนี้:

  • ผู้เล่นที่ไม่มี tag อาชีพที่เกี่ยวข้องจะทำกิจกรรมต่าง ๆ ได้ยากขึ้น เช่น การตัดไม้และขุดแร่ช้าลง
  • เพิ่มความสมจริงและความท้าทายในเกม
  • สร้างความแตกต่างระหว่างอาชีพภายในเกม
  • ส่งเสริมการเลือกเส้นทางการเล่นอย่างมีกลยุทธ์

WEBDEV Help you
WEBDEV Help you เป็นโปรเจกต์เว็บไซต์ที่ผมสร้างขึ้นมาเพื่อเป็นพื้นที่สำหรับฝึกฝนและพัฒนาทักษะด้านการออกแบบและพัฒนาเว็บไซต์ โดยมุ่งเน้นให้เป็นแหล่งเรียนรู้ที่เปิดกว้างสำหรับทั้งตัวผมเองและผู้ที่สนใจ ไม่ว่าจะเป็นผู้เริ่มต้นหรือผู้ที่ต้องการเสริมความรู้ในด้านนี้ โปรเจกต์นี้จะรวบรวมบทความ ตัวอย่างโค้ด เทคนิค และแนวทางการทำงานต่าง ๆ ที่เกี่ยวข้องกับการสร้างเว็บไซต์อย่างมืออาชีพ รวมถึงแบ่งปันประสบการณ์และคำแนะนำที่เป็นประโยชน์ เพื่อช่วยให้ผู้เรียนสามารถเข้าใจและนำไปประยุกต์ใช้ได้จริงในการทำงานหรือโปรเจกต์ของตัวเอง

  • การสร้างและจัดการโปรเจกต์ (Project & Solution Management)
  • การใช้ IntelliSense เพื่อช่วยเติมโค้ดและแนะนำคำสั่งอย่างแม่นยำ
  • การดีบักโค้ด (Debugging) โดยการตั้งเบรกพอยต์และตรวจสอบค่าตัวแปร
  • การแนะนำพื้นฐาน ทางด้าน website
